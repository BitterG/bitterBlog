Granblue Fantasy: Relink（碧蓝幻想：Relink）的三个 Cheat Engine 汇编注入脚本，包含伤害倍数、昏厥控制和角色使用次数修改。

## 伤害倍数

在伤害结算点注入，将单次伤害值乘以自定义倍率。使用 xmm 浮点寄存器进行倍率运算，保持栈 32 字节对齐。

```asm
[ENABLE]
alloc(newmem,2048,"granblue_fantasy_relink.exe"+1B3F798)
label(returnhere)
label(exit)
label(float_multiplier)

newmem:
  sub rsp,20
  movups [rsp],xmm0

  cvtsi2ss xmm0,edx
  mulss xmm0,[float_multiplier]
  cvttss2si edx,xmm0

  movups xmm0,[rsp]
  add rsp,20

  add [rcx+000015B8],edx

exit:
  jmp returnhere

float_multiplier:
  dd (float)2.3

"granblue_fantasy_relink.exe"+1B3F798:
  jmp newmem
  nop
returnhere:

[DISABLE]
"granblue_fantasy_relink.exe"+1B3F798:
  db 89 91 B8 15 00 00

dealloc(newmem)
```

| 注入点 | 功能 | 默认倍率 |
|--------|------|---------|
| `exe+1B3F798` | `add [rcx+15B8],edx` 伤害累加 | 2.3x |

## 昏厥控制

通过 AOB 扫描定位昏厥（stun）值累加指令，支持倍率修改和强制清零两种模式。

- `stunmul_f`: 昏厥倍率，`0` 时跳过，设为 `1.0` 即原始值
- `stun`: 设为 `1` 时强制将昏厥值清零（从 `[rax+04]` 读取 `movss xmm6`）

```asm
[ENABLE]
aobscanmodule(aobstun,granblue_fantasy_relink.exe,
  C4 C1 4A 58 85 ?? ?? 00 00 C4 C1 7A 5D ?? ?? ?? ?? 00 00 C4 C1 7A 11)
alloc(newmem,$1000,aobstun)
alloc(origbytes,16,aobstun)
label(code)
label(return)
label(stun)
label(stunmul_f)
label(disp)
registersymbol(aobstun)
registersymbol(stun)
registersymbol(stunmul_f)

origbytes:
  readmem(aobstun,9)
disp:
  readmem(aobstun+5,4)

newmem:
  push rax
  cmp dword ptr [stunmul_f],0
  je short @f
  mulss xmm6,[stunmul_f]
@@:
  movsxd rax,dword ptr [disp]
  lea rax,[r13+rax]
  cmp dword ptr [stun],1
  jne short code
  movss xmm6,[rax+04]
code:
  vaddss xmm0,xmm6,[rax]
  pop rax
  jmp return

stun:
dd 0
stunmul_f:
dd (float)0

aobstun:
  jmp newmem
  nop 4
return:

[DISABLE]
aobstun:
  readmem(origbytes,9)
unregistersymbol(aobstun)
unregistersymbol(stun)
unregistersymbol(stunmul_f)
dealloc(newmem)
dealloc(origbytes)
```

指令基址为 `[r13+disp32]`，patch 长度 9 字节。预留了 Script B（`[r12+disp32]`，patch 长度 10 字节）以应对游戏版本差异。

## 角色使用次数

在角色结算统计处注入，将 `inc eax`（自增 1）替换为 `mov eax, #1` 后直接写入 `mov [rcx],eax`，固定每次统计为 1 次。这是玩家对局结算记录每个角色使用次数是 1 次，如果你要做其他的内容可以写 `mov eax, #99`。

```asm
[ENABLE]

aobscanmodule(INJECT,granblue_fantasy_relink.exe,
  FF C0 89 01 E8 70 2E DA FE)
alloc(newmem,$1000,INJECT)

label(code)
label(return)

newmem:

code:
  mov eax, #1
  mov [rcx],eax
  call granblue_fantasy_relink.exe+6E23D0
  jmp return

INJECT:
  jmp newmem
  nop 4
return:
registersymbol(INJECT)

[DISABLE]

INJECT:
  db FF C0 89 01 E8 70 2E DA FE

unregistersymbol(INJECT)
dealloc(newmem)
```

> **注入点**: `granblue_fantasy_relink.exe+193F557`

```
原始代码:
  193F557: FF C0       - inc eax
  193F559: 89 01       - mov [rcx],eax
  193F55B: E8 70 2E DA FE - call exe+6E23D0

修改后:
  mov eax, #1       ; 固定为 1 次
  mov [rcx],eax
  call exe+6E23D0
```

## 使用说明

1. 打开 Cheat Engine，附加到 `granblue_fantasy_relink.exe` 进程
2. 将脚本复制到 CE 的 Auto Assemble 窗口（`Ctrl+Alt+A`）
3. 分配内存并注入（`File → Assign to current cheat table`）
4. 勾选启用脚本，修改倍率常量后重新注入生效
