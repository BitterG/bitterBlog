优化 PhasmoCheatV 高亮模式：从 Directional Light 到 Point Light。

> Pull Request: [Switch fullbright from Directional to Point light with range control](https://github.com/ViniLog789/PhasmoCheatV-Recode/pull/XXX)

## 背景

PhasmoCheatV 是一款 Phasmophobia 的辅助工具。原有的高亮模式使用了一个挂载在玩家摄像机上的 **Directional Light**，存在以下问题：

- 灯光位置在玩家脚底，导致**地板完全黑暗**
- Directional Light 无距离衰减，远近一样亮，**不自然**
- 灯光跟随摄像机旋转，转动视角时**光照方向跟着变**
- 不可调节照射范围

## 改动内容

### 1. 光源类型：Directional → Point

```cpp
// Before
SDK::Light_type_set(light, SDK::LightType::Directional, nullptr);

// After
SDK::Light_type_set(light, SDK::LightType::Point, nullptr);
```

Point Light 具有自然的距离衰减（近亮远暗），更接近真实光源的表现。

### 2. 挂载点：摄像机 → 玩家

```cpp
// Before: attached to camera
auto cameraTransform = Utils::GetPlayerTransformCamera(localPlayer);
SDK::Transform_Set_Parent(lightTransform, cameraTransform, nullptr);

// After: attached to player
auto playerTransform = SDK::Component_Get_Transform(
    (SDK::Component*)localPlayer, nullptr);
SDK::Transform_Set_Parent(lightTransform, playerTransform, nullptr);
```

灯光不再随视角旋转，光照方向保持稳定。

### 3. 灯光位置抬高，照亮地板

```cpp
SDK::Vector3 playerPos = Utils::GetPosVec3(localPlayer);
playerPos.Y += 3.5f; // raise above player head so floor is lit
SDK::Transform_Set_Position(lightTransform, playerPos, nullptr);
```

灯光从玩家头顶发出，地板正常被照亮。

### 4. 新增 Range 可调参数

```cpp
// Config
DECLARE_CONFIG(GetConfigManager(), "Range", float, 30.f);

// Apply to light
SDK::Light_range_set(light, range, nullptr);

// UI slider
ImGui::SliderFloat("##Range", &range, 1.f, 100.f, ...);
```

可调范围 1~100，默认 30，用户可根据地图大小自行调整。

### 5. SDK 扩展

`Light.h` 新增 range setter：

```cpp
DEC_MET(Light_range_set, void(*)(Light*, float, MethodInfo*),
    "UnityEngine.CoreModule", "UnityEngine", "Light", "set_range", 1);
```

## 效果对比

![对比图](/img/projects/eb7ce88f7f521237/compare.webp)

| 方面 | 原版 | 优化后 |
|------|------|--------|
| 光源类型 | Directional（无衰减） | Point（距离衰减） |
| 挂载点 | 摄像机 | 玩家 Transform |
| 视角影响 | 跟着转头 | 不受视角影响 |
| 地板 | 完全黑暗 | 正常照亮 |
| 可调参数 | 仅 Intensity | Intensity + Range |
| 强度范围 | 0.1 ~ 100 | 0.1 ~ 10 |
