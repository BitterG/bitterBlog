# 模板

## 1.实现
统一异常处理使用的是 @ControllerAdvice + @ExceptionHandler 来实现的，@ControllerAdvice 表示控制器通知类，@ExceptionHandler 是异常处理器，两个结合表示当出现异常的时候执行某个通知， 也就是执行某个方法事件，具体实现代码如下：

```java
@ControllerAdvice
@ResponseBody
public class MyExceptionHandler {

    private static final Logger log = LoggerFactory.getLogger(MyExceptionHandler.class);

    /**
     * 拦截所有的空指针异常，进行统一的数据返回
     * @param e 异常
     * @return 返回给前端
     */
    @ExceptionHandler(NullPointerException.class)
    public HashMap<String, Object> nullExceptionHandler(NullPointerException e) {
        HashMap<String, Object> map = new HashMap<>();
        map.put("code", "-1");
        map.put("msg", "空指针异常" + e.getMessage());
        map.put("data", null);

        //记录日志
        log.warn("空指针异常{}", e.getMessage());
        StackTraceElement[] stackTrace = e.getStackTrace();
        for (StackTraceElement element : stackTrace) {
            log.warn("{}", element.toString());
        }

        return map;
    }

    /**
     * 兜底异常
     * @param e 异常
     * @return 返回给前端
     */
    @ExceptionHandler(Exception.class)
    public HashMap<String, Object> nullExceptionHandler(Exception e) {
        HashMap<String, Object> map = new HashMap<>();
        map.put("code", "-1");
        map.put("msg", "异常" + e.getMessage());
        map.put("data", null);

        //记录日志
        log.warn("异常{}", e.getMessage());
        StackTraceElement[] stackTrace = e.getStackTrace();
        for (StackTraceElement element : stackTrace) {
            log.warn("{}", element.toString());
        }

        return map;
    }
}
```

## 2.测试
```java
@RestController
@RequestMapping("/")
public class UserController {

    @GetMapping("/getuser")
    public String getUser() {

        //空指针异常
        Object obj = null;
        obj.hashCode();

        System.out.println("执行了 get User~");
        return "get user";
    }

    @GetMapping("/login")
    public String login() {
        System.out.println("执行了 login~");
        return "login~";
    }

    @PostMapping("/reg")
    public String reg() {

        //除零异常
        int num = 10 / 0;

        System.out.println("执行了 reg~");
        return "reg~";
    }

}
```
