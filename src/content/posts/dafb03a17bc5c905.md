# CGLIB多拦截器情况

## 1.创建实类
```java
package DynamicProxy.cglibProxy;

public class FishShop {

    public void buyFish() {
        System.out.println("buy one fish");
    }

}

```

## 2.代理对象工厂类
```java
package DynamicProxy.cglibProxy;

import net.sf.cglib.proxy.Enhancer;
import net.sf.cglib.proxy.MethodInterceptor;

public class CglibProxyFactory {

    private static Enhancer enhancer = new Enhancer();

    public static <T> T getProxyObject(Class<T> clazz, MethodInterceptor[] mInterceptors) {
        // 类加载器
        enhancer.setClassLoader(clazz.getClassLoader());
        // 代理类
        enhancer.setSuperclass(clazz);
        // 拦截器
        enhancer.setCallbacks(mInterceptors);
        // 传入多个回调（拦截器）的情况需要通过过滤器决定使用哪个回调，根据accept函数的返回值（传入拦截器参数的数组下标）
        enhancer.setCallbackFilter((method) -> {
            return 0; //这里返回0，表示使用第一个拦截器
        });

        return clazz.cast(enhancer.create());
    }
}
```

## 3.拦截器类（实现MethodInterceptor）
```java
package DynamicProxy.cglibProxy;

import net.sf.cglib.proxy.MethodInterceptor;
import net.sf.cglib.proxy.MethodProxy;

import java.lang.reflect.Method;

public class BuyFishInterceptor implements MethodInterceptor {
    @Override
    public Object intercept(Object o, Method method, Object[] objects, MethodProxy methodProxy) throws Throwable {
        System.out.println("记录日志");
        Object result = methodProxy.invokeSuper(o, objects);
        System.out.println("打印发票");
        return result;
    }
}
```

第二个拦截器

```java
package DynamicProxy.cglibProxy;

import net.sf.cglib.proxy.MethodInterceptor;
import net.sf.cglib.proxy.MethodProxy;

import java.lang.reflect.Method;

public class BuyFishInterceptor2 implements MethodInterceptor {
    @Override
    public Object intercept(Object o, Method method, Object[] objects, MethodProxy methodProxy) throws Throwable {
        System.out.println("记录日志2");
        Object result = methodProxy.invokeSuper(o, objects);
        System.out.println("打印发票2");
        return result;
    }
}

```

## 4.使用
```java
import DynamicProxy.cglibProxy.BuyFishInterceptor;
import DynamicProxy.cglibProxy.BuyFishInterceptor2;
import DynamicProxy.cglibProxy.CglibProxyFactory;
import DynamicProxy.cglibProxy.FishShop;
import net.sf.cglib.proxy.MethodInterceptor;

public class CglibProxyTest {
    public static void main(String[] args) {
        //这里传入两个拦截器
        FishShop fishShop = CglibProxyFactory.getProxyObject(FishShop.class, new MethodInterceptor[]{new BuyFishInterceptor(), new BuyFishInterceptor2()});
        fishShop.buyFish();
    }
}
```

## 5.总结
:::info
1.使用 enhancer.setCallbacks(拦截器数组对象); 设置多个拦截器

2.需要同时设置enhancer.setCallbackFilter(过滤器对象); 可使用lambda/者匿名内部类/创建类并实现    CallbackFilter接口，重写accept方法返回值表示使用传入拦截器数组的下标

:::
